=============================
Architecture Decision Records
=============================

The following documents the architecture decision records (ADRs) when developing
this project. ADRs are a way to capture architectural decisions made during the
development of a software project. More details can be found in the
:ref:`architecture_decisions_references` section.

.. toctree::
   :maxdepth: 1

   02_inter_thread_comm

A template for ADR is provided at ``templates/decisions.tmpl`` for adding new
ADRs.
